
# Special Reports Country Focus Dashboards

A template a series of local economic dashboards built for [Financial Times Special Reports](https://www.ft.com/special-reports) by [Christopher MÃ¶ller](https://twitter.com/chrtze) and [Moritz Klack](https://twitter.com/moklick) of [Webkid](https://webkid.io/). Currently active uses are dashboard pages about:
